## 一、负责范围

前端 B 负责小程序中与「探索内容」相关的用户端页面，以及管理员数据查看页面：

| 模块 | 页面目录 | 主要职责 |
| :--- | :--- | :--- |
| 故事入口 | `pages/story-mode` | 获取并展示可体验剧情，进入长/短剧情、勋章、存档 |
| 校园长剧情 | `pages/campus-story` | 节点播放、历史回顾、保存进度、完成结算、读档续读 |
| 办事流程 | `pages/process-select`、`pages/process-story` | 流程标记、步骤弹层、短剧情播放与完成结算 |
| 勋章墙 | `pages/badges` | 展示勋章列表、解锁状态与收集数量 |
| 存档管理 | `pages/saves` | 存档列表、读档与删除 |
| 管理员模块 | `pages/admin-login`、`pages/admin-dashboard` | 管理员登录、统计、用户列表、剧情树与导出 |

## 三、已接入接口

| 接口 | 用途 | 调用位置 |
| :--- | :--- | :--- |
| `GET /api/stories` | 获取可用剧情 | `story-mode` |
| `GET /api/stories/:id/nodes` | 获取剧情节点、读档续读 | `campus-story`、`process-story` |
| `POST /api/stories/:id/complete` | 完成剧情、发放勋章 | `campus-story`、`process-story` |
| `GET /api/process-markers` | 获取办事流程、完成状态与步骤 | `process-select` |
| `GET /api/badges/me` | 获取全量勋章及解锁状态 | `badges` |
| `GET /api/save-slots` | 获取当前用户存档 | `saves` |
| `POST /api/save-slots` | 保存长剧情进度 | `campus-story` |
| `DELETE /api/save-slots/:slotId` | 删除存档 | `saves` |
| `POST /api/admin/login` | 管理员登录 | `admin-login` |
| `GET /api/admin/stats` | 获取后台统计 | `admin-dashboard` |
| `GET /api/admin/users` | 获取用户列表 | `admin-dashboard` |
| `GET /api/admin/stories` | 获取后台剧情树 | `admin-dashboard` |
| `GET /api/admin/export/users.xlsx` | 导出用户 Excel | `admin-dashboard` |
| `GET /api/admin/export/stories.txt` | 导出剧情 TXT | `admin-dashboard` |

## 四、联调方式

### 4.1 导入项目

在微信开发者工具导入：

```text
D:\Campus_Explore\rjks_Campus_Tours_for_New_Students\frontend
```

开发者工具本地设置中勾选：

```text
不校验合法域名、web-view、TLS 版本以及 HTTPS 证书
```

### 4.2 临时切换后端地址

默认地址为 `utils/api.js` 中的：

```js
const BASE_URL = 'http://localhost:3000'
```

拿到后端临时公网地址后，在**微信开发者工具 Console** 执行：

```js
wx.setStorageSync('apiBaseUrl', 'https://后端临时地址')
```

查看当前覆盖地址：

```js
wx.getStorageSync('apiBaseUrl')
```

切回默认地址：

```js
wx.removeStorageSync('apiBaseUrl')
```

## 六、注意事项 ⚠️

- 不要提交 `frontend/project.private.config.json`，该文件已在 `.gitignore` 中忽略。
- 不要将临时 ngrok / Cloudflare 地址写入 `utils/api.js` 后提交；使用 `apiBaseUrl` 本地覆盖。
- 用户接口不传 `userId`。后端通过 `Authorization: Bearer <token>` 识别当前用户。
- 管理接口必须使用 `adminToken`；普通用户 Token 访问后台会被拒绝。
- 剧情文本由后端节点接口返回，前端 B 页面不使用旧 `utils/data.js` 中的原型文本。
- 当前素材包中校园、医保背景文件为合并命名文件，路径兼容逻辑已在剧情页处理；如后续补齐单独背景图，应同步更新素材映射。
