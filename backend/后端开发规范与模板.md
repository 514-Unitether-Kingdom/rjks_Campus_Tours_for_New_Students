# 后端新增模块开发指南

## 1. 目录结构规范
- `models/` : 放数据库操作（DAO）
- `controllers/` : 放业务逻辑处理
- `routes/` : 放路由定义
- `middlewares/` : 放中间件（已提供 `auth.js`）

## 2. 编写新模块步骤（以“故事模块”为例）

### 第一步：写 Model（数据层）
在 `models/` 下新建 `Story.js`，参考 `User.js` 的写法：
```javascript
const db = require('../config/db');

const Story = {
  // 查询所有剧情
  async findAll() {
    const [rows] = await db.query('SELECT * FROM stories WHERE status = "enabled"');
    return rows;
  },
  
  // 根据ID查剧情
  async findById(id) {
    const [rows] = await db.query('SELECT * FROM stories WHERE id = ?', [id]);
    return rows[0];
  }
};
module.exports = Story;
