# 《浏览校园》长故事脚本 v2

学姐全程担任向导。每个地点固定生成相邻两页：`scene` 为实景页（当前复用故事背景，后续换实景图），`map` 为地图页（使用同一个 `locationId`，地图放大且地点色块闪烁）。

## 开场与生活区

| 地点 / locationId | 实景页（学姐） |
| --- | --- |
| 校门 / `campus_north_gate` 等 | 欢迎来到北工大平乐园校区！先记住东南西北四个主要校门，以及东南小门、西侧小门；迷路时找校门或体育场，方向就能重新对上。西侧小门离地铁近，但只允许步行。 |
| 北区宿舍 / `north_dormitories` | 01 到 04 号楼离学生综合服务中心很近。先把自己的楼号、单元和门禁位置记牢，早八才不会手忙脚乱。 |
| 东北区宿舍 / `northeast_dormitories` | 10、11 号楼靠近北部教学区和北体育场，出门前预留十分钟，顺路熟悉银杏大道。 |
| 东区宿舍 / `east_dormitories` | 13、14 号楼靠近能源楼和北体育场；晚归时走照明充分的主路，也别忘了门禁时间。 |
| 中部宿舍 / `central_dormitories` | 07 到 12 号楼是生活区中转站，去图书馆、食堂和南体育场都很顺路。 |
| 南区宿舍 / `south_dormitories` | 05、06 号楼靠近南部教学区和奥林匹克体育馆，沿博雅路可以慢慢认清人文楼和科学楼。 |
| 学生综合服务中心 / `student_service_center` | 北区食堂、民族食堂和天天餐厅都在这里。第一次来先认楼层，饭点就不会找不到入口。 |
| 奥运餐厅 / `olympic_restaurant` | 奥运餐厅在 12 号宿舍楼南侧，赶课时优先选择离自己最近的食堂，别为吃什么耽误上课。 |
| 美食园 / `food_court` | 美食园、特色餐厅、教工餐厅和咖啡 1727 集中在餐饮综合楼；晚自习前想补给能量可以来看看。 |
| 快递站 / `express_station` | 快递站在北侧田径场和游泳馆之间，黄色纵向建筑下方的绿色空地就是取件点。快递地址统一填写：北京市朝阳区南磨房路平乐园 100 号。 |
| 校医院 / `school_hospital` | 校医院就在北门门口。可通过“北工大校医院”预约挂号，到院后凭一卡通签到就诊；不舒服时别硬撑，尽早来看看。 |

快递站、校医院的地图页紧接各自实景页，之后才出现分支。食堂时间只在剧情概括提示；完整时间写入“食堂时间徽章”。随后出现选项：**先去学习区** → `study_profile_gate`；**先去运动区** → `sport_north`。

## 运动分支

| 地点 / locationId | 实景页（学姐） |
| --- | --- |
| 北体育场 / `north_sports_ground` | 北体育场是北部最醒目的方向参照。经过训练场地时，记得给上课和校队训练留出空间。 |
| 南体育场 / `south_sports_ground` | 南体育场全天免费开放；跑步前先看有没有教学、比赛或大型活动安排。 |
| 篮球排球场 / `basketball_volleyball_courts` | 球场在南体育场南侧，热门时段建议提前约同学；实际可用情况以预约小程序为准。 |
| 网球场 / `tennis_courts` | 网球场在南体育场北侧，校内外收费不同，练球前先确认预约和当天规则。 |
| 奥林匹克体育馆 / `olympic_venue` | 健身房在体育馆北 4 口附近，羽毛球馆在一层北一门进入；两处都需要留意预约和开放时间。 |
| 乒乓球馆 / `table_tennis_hall` | 乒乓球馆夹在南体育场与西侧南北向道路之间。认准地图上的红色不规则长条，就不会绕圈。 |
| 游泳馆 / `swimming_pool` | 游泳馆在北体育场西侧，首次前按要求办理体检和淋浴卡；开放场次以当天通知为准。 |
| 地下训练馆 / `underground_training_hall` | 地下训练馆在游泳馆北侧，有篮球、排球场地，但会优先保障课程和校队训练。 |

运动时间完整信息写入“运动时间徽章”。分支末尾可选择**再去看看学习区**或**结束浏览**。

## 学习区与专业路线

进入学习区时读取个人资料：`major` 为空则显示“去我的页面填写专业，我会带你走专属路线”；仍可选择通用路线。`major` 非空时以 `college` 优先、专业关键词兜底，选择下表对应地点。

| 地点 / locationId | 实景页（学姐） |
| --- | --- |
| 图书馆 / `library` | 图书馆大致每天 7:30 到 22:30 开放，凭一卡通进入；不同区域规则可直接问工作人员。 |
| 一教 / `no1_teaching_building` | 一教靠近北体育场西侧，数理与部分理工课程常在这里。上课前确认教室号，别只看楼名。 |
| 二教 / `no2_teaching_building` | 二教与一教相连，材料楼在南侧；赶课时预留上下楼和找教室的时间。 |
| 三教 / `no3_teaching_building` | 三教在图书馆东北侧，是许多专业都可能用到的公共教学楼。 |
| 四教 / `no4_teaching_building` | 四教靠近南体育场，文科和艺术相关课程较常出现，也可顺路熟悉艺术广场。 |
| 机电楼 / `mechanical_electrical_building` | 机械、智能制造和能源动力相关同学会从这里接触设备、结构与工程实践。 |
| 智研楼 / `intelligent_research_building` | 智研楼在一教西南侧；做项目讨论时，记住它和图书馆北侧的相对位置。 |
| 金工楼 / `goldworking_building` | 金工楼在北体育场北侧；实践课进入前要遵守实验和设备安全要求。 |
| 绿建楼 / `green_building` | 绿建楼在礼堂北侧，建筑、绿色低碳等话题会在这里出现。 |
| 能源楼 / `energy_building` | 能源楼在北体育场东南侧，环境、新能源和机械能源方向都会与它产生交集。 |
| 暖通楼 / `heating_ventilation_building` | 暖通楼在北体育场东侧、能源楼北侧，别把两座相邻楼混淆。 |
| 材料楼 / `material_building` | 材料楼在二教南侧、图书馆北侧，是材料方向认识新材料与工程应用的重要地点。 |
| 信息楼、信息楼东 / `information_building`、`information_building_east` | 信息楼在图书馆西侧；信息楼东是 C 形结构中包裹的小长方形楼，找教室时尤其留意。 |
| 实训楼 / `practice_training_building` | 实训楼在月亮湖东侧，信息和计算机相关同学可在这里接触实践训练场景。 |
| 建工楼、城建楼 / `construction_building`、`urban_construction_building` | 建工楼与城建楼连接建筑工程、建筑规划和创新实践；城建楼位于东南边缘教学区。 |
| 城建楼、实训楼 / `urban_construction_building`、`practice_training_building` | 城市交通方向在这里接触智慧城市、交通规划与实践训练。 |
| 环能楼、生命楼 / `environment_energy_building`、`life_science_building` | 环能楼在生命楼南侧；环境工程、生命科学与医工交叉方向会在这里展开。 |
| 数理楼 / `physical_science_building` | 数理楼在图书馆北侧、游泳馆西侧，是物理光电、数学统计和力学的重要坐标。 |
| 人文楼、经管楼、科学楼 / `humanities_building`、`economics_management_building`、`science_building` | 人文、外语、社会、经管等方向在南部教学区展开；三栋楼彼此相邻，适合一次认清。 |
| 艺术楼 / `art_building` | 艺术楼在四教南侧、艺术广场中心，艺术设计同学可以从这里熟悉创作与展示空间。 |
| 软件楼 / `software_building` | 软件楼在月亮湖东侧，是人工智能、网络安全等方向的另一处重要学习地点。 |
| 知新园 / `zhixin_garden` | 知新园在图书馆东侧，是图书馆周边行走、会合和短暂停留的好参照。 |

每行均紧接一张地图页，统一文案为：“看这片闪烁的金色区域，它就是 **{地点}**；从 **{相邻地标}** 出发，很快就能找到入口。”
